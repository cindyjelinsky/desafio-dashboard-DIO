# Projeto: Dashboard em Excel-DIO

## Descrição do Projeto
Este projeto consiste na criação de um **dashboard interativo** a partir de uma base de dados fornecida, com o objetivo de facilitar a análise de informações e gerar insights de forma visual e organizada.  
O dashboard foi desenvolvido como parte de um desafio de análise de dados utilizando ferramentas de planilhas eletrônicas.
